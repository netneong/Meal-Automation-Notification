# 대학교 기숙사 식단 알람_디스코드 연결

## ✔️요구사항
- 문제: 기숙사 식단의 확인 과정이 복잡함, 스마트폰 알람탭에서 확인불가
- 개인 사용목적이 아닌 기숙사 친구들과 공유가능해야함
## ✔️설계
- 대학교 기숙사 사이트 확인 결과 HTML로 식단을 파싱할 수 있다는 것을 확인
- BeautifulSoup를 이용해서 정보를 파싱하는것으로 계획
- 디스코드 채널과 github actions를 이용해서 서버 비용없이 사용할 수 있도록 계획
- 하루 식단 전체, 각 식단시간별로 알람을 보내도록 계획 -> 가독성 측면에서 하루 식단 전체만 사용하도록 변경
## ✔️유지보수
- 디스코드 알람이 1시간 가량 지연이 발생하는 것을 확인
-> github actions의 지연으로 쓰는 사용자가 많은 경우 지연이 발생할 수 있다는 것을 확인(시간 조정)
## ✔️기능별 동작
```
[GitHub Actions (KST 06:58)] 
       │
       ▼
[BeautifulSoup4 스크래퍼 구동] ➔ "조식/아침" 키워드로 식단 Table 추출
       │
       ▼
[동적 인덱싱] ➔ 오늘 요일 칸 위치 파악 후 메뉴 텍스트 정제
       │
       ▼
[Discord Webhook] ➔ 포맷팅된 텍스트 채널 전송 후 서버 즉시 종료
```

- 트리거 단계: KST 06:58(UTC 전날 21:58)에 GitHub Actions 러너가 구동

- 데이터 수집: requests로 HTML을 받아온 후, find(string=...)을 통해 식단표의 시작점을 타겟팅

- 데이터 정제: 추출된 텍스트에서 불필요한 공백과 "식단이미지" 등의 노이즈 단어를 필터링하고 배열로 구조화

- 알림 발송: 보안 주입된 WEBHOOK_URL 환경 변수를 통해 디스코드로 데이터를 전송
